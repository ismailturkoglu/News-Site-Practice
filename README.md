# News Site - Practice Project

A responsive news website built with **HTML** and **CSS**, focusing on modern layout techniques and clean UI design.

[**https://ismailturkoglu.github.io/news-site-practice/**](https://ismailturkoglu.github.io/news-site-practice/)

# Features

* Fully responsive design (mobile, tablet, desktop)
* CSS Grid based layout system
* Reusable utility classes (spacing, colors, buttons)
* Semantic HTML structure
* Modern UI with category tags and cards
* Article detail page with sidebar

# Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Google Fonts
* Font Awesome

# Responsive Breakpoints

* Mobile:           ≤ 360px
* Small Devices:    ≤ 576px
* Tablet:           ≤ 768px
*  Desktop:         > 768px

# What I Learned

* How to build responsive layouts using **CSS Grid**
* Creating reusable CSS utility classes
* Structuring scalable CSS with variables (:root)
* Mobile-first thinking and media queries
* Layout design for real-world UI
